# agent-rules

Universal engineering standards for building production-grade AI applications.

**For whom:** AI agents (Claude, GPT, Gemini, Mistral) + human developers

**What:** 14 files covering:
- Engineering constitution & reasoning framework
- Naming conventions for predictive navigation
- Feature architecture & full-stack workflows
- API contracts, error handling, resilience
- Database migrations & tenant isolation
- UI/UX design system & state management
- Security, dependencies, startup protocols
- LLM integration with guardrails
- Deployment, rollback, monitoring
- AI maintenance protocol (diagnosis, modification, refactoring)

**Why:** 
- Code that is correct, secure, operable, and auditable by default
- AI agents can maintain without human oversight
- Features scale to hundreds of files with predictable structure
- Errors are handled consistently across all services
- Every operation is idempotent and safe to retry

**Get started:** Read `README.md` for the complete index and protocol.
